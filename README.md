# FRO
Minimalistic Frogger clone in less than 500 bytes
## Tech
* ZX Spectrum 48
* Z80 Assembler ( <500 bytes executable size)
## Screenshot
![Screenshot](fro.png)
## Controlls
* Kempston OR Sinclair 1 / Keyboard (keys 1,2,3,4)
